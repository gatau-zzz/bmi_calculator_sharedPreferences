# Praktikum 5 – Persistensi Data dengan SharedPreferences

Ganjar Putra Perkasa (085023005)

## Struktur Proyek & Setup

### Menambahkan Dependency
1. Pastikan sudah menggunakan Flutter SDK ≥ 3.0
2. Masuk ke direktori bmi_calculator
3. Tambahkan (shared_preferences: ^2.0.15) di pubspec.yaml
4. Jalankan flutter pub get

### Membuat StorageService
5. Buat file baru lib/services/storage_service.dart: dan masukan kodingannya

### Integrasi SharedPreferences di UI
6. Modifikasi lib/main.dart — fokus pada _BmiHomeState dan BmiResult:
7. Jalankan project dengan flutter run
